# qinglonglu.github.io

陆怡安的个人笔记

## 项目简介

这是一个清新淡雅的个人笔记网站，采用绿色为主色调，包含文章展示、个人介绍等功能。

## 网站地址

[https://qinglonglu.github.io](https://qinglonglu.github.io)

## 功能特性

### 页面结构
- **首页**：展示最新3篇文章
- **关于我**：个人介绍页面
- **我的文章**：所有文章列表页
- **文章详情页**：单篇文章展示页

### 核心功能
1. **文章展示**：3段式结构
2. **搜索功能**：支持单字匹配搜索，匹配字标红高亮
3. **响应式设计**：适配各种设备屏幕
4. **交互效果**：鼠标悬停时按钮有弹跳动画
